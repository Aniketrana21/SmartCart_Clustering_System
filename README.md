# SmartCart_Clustering_System
SmartCart Customer Segmentation System using Machine Learning to analyze customer behavior, perform user characterization, and discover meaningful customer groups using K-Means and Agglomerative Clustering.

🛒 SmartCart Customer Segmentation System

An end-to-end Machine Learning project that analyzes customer demographics, purchasing behavior, and engagement metrics to identify distinct customer segments. The project applies K-Means and Agglomerative Clustering algorithms, along with EDA using NumPy, Pandas, Matplotlib, and Seaborn, to perform customer characterization and support data-driven marketing decisions.

🚀 Key Features
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Customer Behavior Analysis
User Characterization
K-Means Clustering
Agglomerative Hierarchical Clustering
Cluster Visualization
Business Insights & Recommendations
🛠️ Tech Stack
Python
NumPy
Pandas
Matplotlib
Seaborn
Scikit-Learn
📊 Objective

To segment customers into meaningful groups based on demographics, spending habits, purchase frequency, and engagement patterns, enabling personalized marketing and customer retention strategies.
